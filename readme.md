점-선-면-차원 인문학적 프레임워크
🔴
점 (Point) - 순간과 존재
일상의 작은 순간들에서 의미 발견하기

인문학적 질문
한 사람이 공간에서 경험하는 ‘지금 이 순간’의 의미는 무엇인가?
디지털 기기와의 접촉점(터치, 클릭, 시선)이 우리 존재에 미치는 영향은?
센서가 포착하는 ‘데이터 포인트’와 인간이 느끼는 ‘경험의 순간’의 간극은?
연구 관점
현상학적 접근: 공간 속 신체의 경험 | 심리학적 접근: 미시적 감정과 인지

➖
선 (Line) - 흐름과 관계
움직임 속에서 드러나는 삶의 서사

인문학적 질문
한 사람의 동선은 단순한 이동인가, 삶의 이야기인가?
출근길, 퇴근길의 반복이 만드는 정체성은 무엇인가?
디지털 연결망(SNS, 메신저)이 물리적 동선과 어떻게 교차하는가?
연구 관점
서사학적 접근: 일상의 내러티브 | 사회학적 접근: 관계의 동역학

▭
면 (Plane) - 장소와 공동체
공간이 품은 문화, 사람들이 만드는 의미

인문학적 질문
‘집’은 단순히 거주하는 공간인가, 정체성을 구성하는 장소인가?
공유 공간(카페, 도서관, 공원)은 어떻게 공동체를 형성하는가?
디지털 플랫폼(Zoom 회의실, 메타버스)도 ‘장소’가 될 수 있는가?
연구 관점
인류학적 접근: 공간의 문화적 의미 | 철학적 접근: 장소성과 정주

📦
차원 (Dimension) - 시간과 의미
과거-현재-미래를 가로지르는 삶의 층위

인문학적 질문
AI가 예측한 ‘미래의 나’와 내가 꿈꾸는 ‘이상적인 나’는 같은가?
기술로 최적화된 삶은 정말 ‘좋은 삶(eudaimonia)’인가?
과거의 기억, 현재의 경험, 미래의 가능성이 공간에서 어떻게 중첩되는가?
연구 관점
시간론적 접근: 다층적 시간 경험 | 윤리학적 접근: 기술과 좋은 삶
